# Phase 2D 호환 가져오기

## 상태

- 상태: completed
- 생성일: 2026-07-21
- 마지막 갱신: 2026-07-21
- 현재 하위 단계: 완료

## 목표

- Codex legacy v1 및 v2 atlas를 MonglePet 내부 manifest로 안전하게 변환한다.
- GIF, APNG와 명시적으로 정렬된 PNG 시퀀스를 `idle` 모션 패키지로 변환한다.
- 모든 호환 입력을 기존 `PetPackageLoader`가 다시 검증할 수 있는 데이터 전용 `.monglepet` 디렉터리로 만든다.

## 범위

- Codex `pet.json`과 atlas 버전·크기·경로 검증
- Codex 표준 9개 행과 v2 방향 16개 셀의 모션 매핑
- 사용 셀 비어 있음 및 미사용 셀 불투명 픽셀 검증
- GIF/APNG 프레임과 표시 시간 해석
- 호출자가 순서를 확정한 PNG 파일 목록의 단일 모션 변환
- 프레임을 투명 PNG atlas와 `preview.png`로 정규화
- 생성 결과를 기존 패키지 로더로 다시 검증하고 실패 시 출력 정리
- 실제 이미지 기반 adapter 단위 및 임시 디렉터리 통합 테스트

## 제외 범위

- `NSOpenPanel`, 가져오기 확인 화면과 파일 순서 편집 UI
- 가져온 패키지의 즉시 Application Support 설치 및 중복 선택 UI
- animated WebP 단일 파일 가져오기
- Codex v2 방향 셀을 포인터 추적 런타임에 직접 연결하는 기능
- 타사 캐릭터 자산의 재배포 또는 저장소 포함

## 열린 질문

- 없음

## 결정사항

- adapter 출력은 별도 런타임 모델이 아니라 기존 스키마 버전 1 `.monglepet` 디렉터리다.
- Codex manifest의 버전과 실제 atlas 크기가 다르면 크기로 자동 보정하지 않고 거부한다.
- Codex atlas의 사용 셀은 하나 이상의 불투명 픽셀이 있어야 하고 미사용 셀은 완전히 투명해야 한다.
- v2 방향은 `000`을 위쪽으로 시작하는 시계 방향 16개 단일 프레임 모션으로 모두 보존한다.
- GIF/APNG는 ImageIO가 제공하는 프레임별 합성 결과와 지연 시간을 사용하고, 누락되거나 지나치게 짧은 시간에는 안전한 기본값을 적용한다.
- PNG 시퀀스 순서는 파일명으로 암묵 추정하지 않고 호출자가 전달한 순서를 그대로 사용한다.
- 간편 가져오기는 하나의 `idle` 반복 모션을 만들며 제작자와 라이선스는 `Unknown`으로 기록한다.
