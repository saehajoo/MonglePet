## 목표

- 디렉터리 형태 `.monglepet`의 `pet.json`을 안전하게 읽는다.
- manifest, 경로, 이미지와 모션 데이터를 설치 전에 전체 검증한다.
- 검증된 패키지를 기존 `PetDefinition` 런타임 모델로 변환한다.
- 잘못되거나 과도한 패키지를 명확한 오류로 거부한다.

## 범위

- 패키지 메타데이터, atlas와 미리보기 리소스 모델
- 패키지 내부 안전한 상대 경로와 심볼릭 링크 검증

- atlas·모션 ID, 기본 모션, 프레임 범위·시간과 전체 개수 검증
- `PetDefinition` 변환과 임시 디렉터리 기반 단위 테스트

## 결정사항

- `PackageLoader`는 읽기와 검증만 수행하고 원본 디렉터리를 수정하거나 설치하지 않는다.
- manifest DTO는 파일 형식에만 사용하고 앱 실행에는 검증 후 생성한 Domain 모델을 사용한다.
- 패키지의 모든 파일은 JSON, PNG 또는 WebP 데이터만 허용하며 심볼릭 링크는 거부한다.
- 실제 이미지 형식은 확장자와 ImageIO 결과가 모두 일치해야 한다.
- atlas는 단일 프레임 이미지와 알파 채널을 요구하며 선언 크기와 실제 픽셀 크기가 일치해야 한다.
- 기본 상한은 압축 해제 크기 100 MiB, 이미지 한 변 8192 px, 전체 디코딩 픽셀 64 MiPixels, 모션 100개와 프레임 1,000개다.
- `defaultMotion`이 생략되면 `idle`을 사용하고 최종 기본 모션이 없으면 패키지를 거부한다.
