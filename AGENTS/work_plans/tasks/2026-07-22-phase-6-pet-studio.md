# Phase 6 펫 스튜디오와 메타데이터 편집

## 상태

- 상태: done
- 생성일: 2026-07-22
- 마지막 갱신: 2026-07-22

## 목표

- 일반 설정에서 현재 펫의 모습, 정보와 전체 애니메이션을 한곳에서 확인한다.
- MonglePet 사용자 펫의 메타데이터와 애니메이션을 안전하게 수정·삭제한다.
- 외부에서 가져온 패키지는 편집 가능한 사본을 만든 뒤 수정한다.

## 범위

- 현재 펫 미리보기와 애니메이션 선택 재생
- 이름, 제작자, 버전, 설명, 라이선스와 기본 애니메이션 편집
- 애니메이션 수정 진입점과 삭제
- 마지막·기본 애니메이션 삭제 보호
- 외부 패키지의 새 ID·설치 UUID 편집 사본 생성
- 임시 사본 검증과 같은 설치 UUID 원자적 교체
- 삭제 애니메이션을 참조하는 현재 펫 행동 설정 복구 경계 준비

## 제외 범위

- 프레임별 시간 편집과 행동 모델 v2
- `.monglepet` 내보내기
- 온라인 카탈로그
- 내장 몽글이 자산 수정·삭제

## 열린 질문

- 없음

## 결정사항

- 패키지 ID는 수정하지 않는다.
- 사용자 펫 정보 편집은 현재 설치 UUID를 유지한다.
- 가져온 패키지는 직접 수정하지 않고 편집 가능한 사본을 만든다.
- 기본 애니메이션과 마지막 애니메이션은 안전한 대체값 없이 삭제할 수 없다.

## 작업 순서

- [x] 1단계: 펫 상세·애니메이션 미리보기와 재생 리소스 연결
- [x] 2단계: 메타데이터·기본 애니메이션 원자적 편집
- [x] 3단계: 애니메이션 수정·삭제와 참조 복구
- [x] 4단계: 편집 가능한 사본 만들기
- [x] 5단계: 일반 설정 UI, 테스트와 실제 앱 QA

## 검증 방법

- 현재 펫과 선택 애니메이션의 실제 프레임 재생을 확인한다.
- 이름·제작자·버전 수정 후 재실행과 라이브러리 재로드 결과를 확인한다.
- 기본·마지막 애니메이션 삭제가 차단되는지 확인한다.
- 편집 실패가 기존 설치 파일을 바꾸지 않는지 확인한다.
- 가져온 패키지의 원본 ID·설치가 유지되고 편집 사본만 변경되는지 확인한다.
