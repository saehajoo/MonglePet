- 디렉터리형 `.monglepet` manifest, 경로, PNG·정적 WebP, atlas와 모션 전체 검증 로더 구현 완료

- Codex legacy v1/v2, GIF, APNG와 명시적 PNG 시퀀스를 내부 패키지로 변환하는 adapter 구현 완료
- 자동·수동 결정 우선순위, 유휴 히스테리시스와 시간 기반 행동 목록 스케줄러 구현 완료
- 단계 경계 전환, 일회성 상호작용 중단·복귀, cooldown과 누락 모션 `idle` 폴백 구현 완료
- Debug 빌드와 단위 테스트 76개 통과, 전체 화면·Space·다중 모니터와 모션 재생을 포함한 실제 앱 수동 QA 완료
- 정적 펫 표시 상태 기준 CPU 0.0%, 메모리 약 26MB 측정
- 임시 내장 모션 표시 상태 CPU 0.0%, 메모리 약 24MB 측정
- 프로젝트 지침과 상세 문서를 `AGENTS.md` 및 `AGENTS/` 구조로 정리

## 1단계: 화면에 보이는 최소 제품

### 1A: 앱 셸

- 기본 SwiftUI `WindowGroup` 제거
- `AppCoordinator` 기반 앱 생명주기 구성
- AppKit `NSStatusItem` 메뉴 막대 항목 추가
- Dock 아이콘을 표시하지 않는 agent-style 앱 설정
- 설정 열기와 앱 종료 진입점 추가

완료 조건: 일반 앱 창이나 Dock 아이콘 없이 메뉴 막대에서 MonglePet을 제어하고 종료할 수 있다.

### 1B: 펫 오버레이

- 투명하고 테두리가 없는 non-activating `NSPanel`
- 교체 가능한 임시 투명 PNG 표시
- 일반 앱 위, 모든 Space와 전체 화면 보조 공간에 표시
- 화면 보호기 및 잠금 화면보다 위에는 표시하지 않음

완료 조건: 현재 작업 중인 앱의 활성 상태를 빼앗지 않고 임시 몽글이가 안정적으로 표시된다.

### 1C: 기본 상호작용

- 펫 드래그
- 깨우기와 재우기
- 디스플레이 구성 변경 시 화면 밖 위치 보정
- 메뉴 막대에서 숨겨진 펫을 다시 깨울 수 있는 복구 경로 유지

완료 조건: 임시 몽글이를 이동하고 숨겼다가 메뉴 막대에서 다시 표시할 수 있다.

### 1D: 검증

- 표시 상태 전환 단위 테스트
- 메뉴 막대 및 앱 실행 UI 스모크 테스트
- 다중 모니터, 전체 화면과 여러 Space 수동 확인
- 정지 상태 CPU 기준선 측정 및 기록

단계 완료 조건: 임시 몽글이 이미지가 안정적으로 표시·이동·숨김 처리되고 현재 작업 앱을 방해하지 않는다.

## 2단계: 펫 패키지와 모션 재생

### 2A: 모션 모델과 내장 펫 재생

- `PetDefinition`, `Motion`과 프레임 모델
- 기본 펫 번들
- 프레임 캐시와 애니메이션 재생기
- 정지, 숨김과 화면 잠금 상태에서 프레임 시계 중지

### 2B: MonglePet 패키지 로더

- 디렉터리 형태 `.monglepet` manifest 파서
- 이미지, atlas, 프레임 범위와 시간 검증
- 잘못된 패키지 단위 테스트

### 2C: 외부 패키지 설치

- ZIP 컨테이너 `.monglepet` 압축 해제 방식 확정
- App Sandbox 사용자 선택 파일 접근
- 임시 위치 전체 검증 후 Application Support에 원자적 설치
- 중복 패키지 업데이트 및 별도 사본 처리

### 2D: 호환 가져오기

- Codex legacy v1 8×9 WebP 아틀라스 adapter
- Codex v2 8×11 WebP 아틀라스 adapter
- GIF, APNG와 PNG 시퀀스 adapter
- 호환 입력을 MonglePet manifest로 변환
- animated WebP 단일 파일은 MVP에서 제외

완료 조건: 기본 펫과 가져온 테스트 펫의 여러 모션을 선택해 재생할 수 있다.

## 6단계: 성능과 배포

- Instruments 기준선 측정
- 화면 잠금과 재우기 상태의 렌더링 중지 확인
- 1시간 이상 메모리 증가 검사
- 다중 모니터, Retina, 전체 화면, 여러 Space QA
- 로그인 시 실행, 앱 서명, notarization 검토
- 개인정보 및 오픈소스 고지 작성

초기 목표:

- 펫 숨김·정지 상태 평균 CPU: 측정 오차에 가까운 수준
- 일반 애니메이션 평균 CPU: 1% 미만 목표
- 설정창을 닫은 뒤 관련 UI 메모리 해제
- 1시간 실행 후 지속적인 메모리 증가 없음

## 7단계: Windows 별도 구현
